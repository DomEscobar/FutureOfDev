# Trae & Pulumi: The 2026 Specialists Audit

## target-company-size: 1,000 Employees ($1B+ Mid-Tier)

---

## 1. Trae (ByteDance)
### Vibe: The High-Velocity Challenger
*   **Background:** Developed by **ByteDance**. Built as an AI-native IDE (VS Code fork) focused on "Conversational Coding" and high-speed autonomy.
*   **Enterprise Pricing (Mid-Tier):**
    *   **Tier:** "Trae Pro" launching with aggressive pricing (referenced as $3-$15/mo for introductory tiers). 
    *   **Enterprise:** Custom tier. Focuses on **Unlimited GPT-4o and Claude 3.7 access** within a modernized UI.
    *   *Source:* [Trae.ai Pricing](https://www.trae.ai/pricing) / [AI Cloudbase Review](https://aicloudbase.com/tool/trae)
*   **Sovereignty & Security (Score: 3/5):**
    *   **Compliance:** SOC 2 efforts underway, but currently lacks the maturity of GitHub or Anthropic for 1k EMP enterprises.
    *   **Data Collection:** Extensive telemetry system noted in technical reviews. Enterprise tiers claim Zero Data Retention (ZDR), but the parent company (ByteDance) origin remains a point of scrutiny for many mid-tier US/EU firms.
    *   *Source:* [Unit221B Technical Report](https://blog.unit221b.com/unveiling-trae-bytedances-ai-ide-and-its-extensive-data-collection-system/)
*   **Flexibility & Vendor Lock (Score: 3/5):**
    *   **IDE Support:** **Hard Fork.** Like Cursor/Windsurf, you must migrate your team to the Trae IDE to utilize "Builder Mode."
    *   **Vendor Lock-in:** High IDE lock-in. However, it boasts high plugin compatibility with the existing VS Code ecosystem.
*   **Agentic Capability:** 
    *   **Builder Mode:** Specialized for full-stack project generation and autonomous error correction.
    *   *Source:* [Trae IDE News](https://traeide.com/news/1)

## 2. Pulumi AI (Neo Agent)
### Vibe: The Infrastructure-as-Agent Specialist
*   **Vibe:** Specializes in **DevOps and Platform Engineering**. It is not an IDE, but an "AI Infrastructure Agent" named **Neo**.
*   **Enterprise Pricing:**
    *   **Tier:** "Cloud Enterprise". pricing based on resource counts and usage.
    *   *Source:* [Pulumi Pricing](https://www.pulumi.com/pricing/)
*   **Data Sovereignty (Score: 5/5):**
    *   **Compliance:** SOC 2 Type II certified.
    *   **Deployment:** Offers a **Self-Hosted / Managed Cloud** option, making it safe for high-compliance 1k EMP companies to manage infrastructure via AI.
    *   *Source:* [Pulumi Enterprise Edition (AWS Marketplace)](https://aws.amazon.com/marketplace/pp/prodview-dwn22batkhsyg)
*   **Flexibility & Vendor Lock (Score: 5/5):**
    *   **IDE Support:** Agnostic. Works via **CLI and Plugins** (VS Code, JetBrains, etc.). No hard fork required.
    *   **Vendor Lock-in:** **Low.** Multi-cloud by design (AWS, Azure, GCP, K8s).
*   **Agentic Capability:**
    *   **Neo Agent:** Can autonomously provision, govern, and optimize infrastructure. Specialized for "Superintelligence Infrastructure" that scales alongside AI workloads.
    *   *Source:* [Pulumi Neo Product](https://www.pulumi.com/product/neo/)
