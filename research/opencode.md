# OpenCode Research

## Vibe: The Open-Source Infrastructure Standard

## Last Verified: 2026-02-15 11:20

## Key Facts:
*   **Enterprise Pricing (Mid-Tier 1k EMP):**
    *   **Core Software:** **Free & Open Source** (MIT License).
    *   **Enterprise Tier:** Custom. Focuses on **SSO integration, internal AI gateway delivery, and dedicated support**.
    *   **Zen Managed Models:** A "Pay-as-you-go" service ($20 baseline + auto-topup) with zero markups on curated models specifically benchmarked for agents. 
    *   *Source:* [OpenCode.ai Enterprise](https://opencode.ai/enterprise) / [OpenCode.ai Zen](https://opencode.ai/zen)
*   **Data Sovereignty (Score: 5/5):**
    *   **Privacy Model:** 100% open source. Organizations can clone and deploy their own internal instance.
    *   **No Retention:** Explicitly promises zero data/context storage.
    *   **Self-Hosting:** Truly self-hostable in private infrastructure. 
    *   *Source:* [OpenCode.ai Landing](https://opencode.ai/)
*   **Vendor Lock-in (Zero Risk):**
    *   **Model Arbitrage:** Extreme. Supports 75+ LLM providers. Compatible with existing ChatGPT Plus, Claude Pro, and GitHub Copilot accounts.
    *   **Open Source:** Full access to the codebase (TypeScript/Node.js). No binary lock-in.
    *   *Source:* [OpenCode.ai GitHub](https://github.com/anomalyco/opencode)
*   **Flexibility (Score: 5/5):**
    *   **IDE Support:** Available as a **Terminal interface, Desktop app, and IDE extension**.
    *   **Plugin Status:** Can be used as a plugin for various editors, avoiding the "Hard Fork" requirement.
    *   *Source:* [OpenCode.ai Docs](https://opencode.ai/docs)
*   **Agentic Capability (Score: 5/5):**
    *   **Tooling Access:** Built-in agents (**build** and **plan**) with out-of-the-box LSP support and terminal/bash execution.
    *   **Parallelization:** Supports **Multi-session** mode where multiple agents run in parallel on the same project.
    *   **Workflow Integration:** Supports Agent Skills and local context files (`AGENTS.md`).
    *   *Source:* [OpenCode GitHub - Agents](https://github.com/anomalyco/opencode#agents)
*   **Stack Transparency (OSS Tech Stack):**
    *   **Language:** Primarily **TypeScript** (50.7%) and **MDX** (45.2%).
    *   **Runtime:** Node.js / Bun. Uses **Turbo** for build orchestration and **SST** (Serverless Stack) for some infrastructure components.
    *   **Orchestration:** Client/Server architecture allowing remote driving of the TUI from other clients.
    *   *Source:* [OpenCode GitHub Languages](https://github.com/anomalyco/opencode)
*   **Sustainability & Longevity:**
    *   **Community:** Significant momentum with **105,000 GitHub Stars**, 700+ contributors, and 2.5M monthly active developers.
    *   *Source:* [OpenCode.ai Metrics](https://opencode.ai/)
